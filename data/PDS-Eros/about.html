Map Projection and Arc-GIS

Data can be imported into ArcMAP 10.1 by downloading the ISIS Map cubes.
All images (PNG and thumbnails) were generated with an equirectangular projection.

Data processing

We used the USGS software tool Integrated Software for Imagers and Spectrometers (ISIS) as the main tool to process the data. We used the shape model posted at USGS and the ISIS software routines to convert images into map-based projections.

Coordinates

All data for the MSI use the Western Longitude System. To comply with the IAU Standard, the correct longitude is 360 deg West longitude. Some other instruments follow a different convention, so it is highly recommended to consult this document for details on the coordinate system.
